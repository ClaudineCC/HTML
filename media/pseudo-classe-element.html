Trabalhando com Pseudo-Classes

São usadas para definir um elemento com um estado especial

Serve para:

  • Estilizar um elemento quando o usuário passa o mouse sobre ele.
  • Estilizar o link visitado e o não visitado de forma diferente.
  • Estilizar um elemento quando ele recebe o foco.

Pseudo-Classe


Comportamentos:

: LINK link a ser clicado(colocar em um navegador diferente para testar)

: VISITED comportamento depois de clicado

: HOVER comportamento de passar o mouse

: ACTIVE comportamento de link ativo: clicar e segurar
